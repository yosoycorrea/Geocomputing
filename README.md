# Framework de Geocomputing para Territorios Híbridos

## Propósito
Analizar y documentar dinámicas espaciales en ciudades tfronterizas mediante herramientas abiertas.

## Componentes
- Datos: mapas, censos, testimonios.
- Herramientas: QGIS, Python, GitHub.
- Procesos: simulación de movilidad, análisis ambiental.
- Resultados: mapas críticos, memoria institucional.
